##### 대구광역시 불법주정차 단속차량 최적경로 선정
 * 팀 프로젝트 (6인)
 * 수행 기간 : 2022.07 ~ 2022.08 (약 3주)
 * 내용 : 회귀 모델링, 이진 분류
 * 역할 : 데이터 전처리, 모델링, PPT작성
 * 기술 : 언어 - Python, R / 툴 - Colab, RStudio, QGIS


##### 연구 주요내용
 1. 주정차 위반 다발 구역을 분석하여 불법 주정차에 영향을 미치는 요인 파악
 2. 선제적으로 단속이 필요한 구역 선정, 각 구청에서 출발하는 단속차량의 최적 경로 개발
